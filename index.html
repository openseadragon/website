BUILT IN

Sixteen years of details, ready out of the box.

No gradient compromises. No "but only if you also load…". The full set, tested across every browser that matters.

Tile sources

DZI, IIIF, OSM-style, Zoomify, Legacy, or your own subclass. Multi-resolution pyramids load on demand.

Overlays

Anchor DOM, SVG, or canvas elements in image coordinates. They track the user's pan, zoom and rotation perfectly.

Navigator

A minimap that mirrors the main view. Configurable, draggable, and aware of multi-image worlds.

Rotation

Free rotation by degree, with flip and constrained-axis modes. Animates with the same easing the viewport uses.

Multi-image

Tile any number of images in one world. Compose collections, stitch composites, or wire sequence-mode for slideshows.

Plugin ecosystem

Filters, annotations, scalebar, screenshots, fabric.js, SVG overlay, GeoJSON — drop them in.

WebGL drawer

Opt-in WebGL rendering for smoother filters, blend modes, and large compositions on capable hardware.

Mobile gestures

Pinch, double-tap, two-finger pan, momentum. Tuned for touch first; keyboard-accessible second.

SIX LINES

Same code as the demo on the right.

Drop one element. Point at a tile source. You're done. Everything else is opt-in.

viewer.html esm · 18 kB 
// 1. include the script
<script src="openseadragon.min.js"></script>

// 2. drop a container
<div id="viewer" style="width:100%; height:600px"></div>

// 3. point at a tile source
const viewer = OpenSeadragon({
  id: "viewer",
  tileSources: "/tiles/painting.dzi",
  showNavigator: true,
  zoomPerScroll: 1.4
});
